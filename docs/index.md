# Herzlich Willkommen!

Sie sind eine öffentliche Einrichtung und suchen nach einem Ticketing-System?
Dann sind Sie hier genau richtig.
Auf dieser Website finden Sie einen Muster-Leistungskatalog, den Sie als Ausgangspunkt für Ihre Ausschreibung verwenden können.
So sparen Sie viel Zeit und können sichergehen, dass Sie keine wesentlichen Aspekte bei der Beschaffung des Ticketing-Systems übersehen haben.
Unser Muster-Leistungskatalog geht hierbei auf die Bedürfnisse von fünf Arten öffentlicher Einrichtungen besonders ein.
Wählen Sie für weitere Informationen aus, welcher Bereich am besten auf Sie zutrifft:

- [Messegesellschaften](messe/index.md)
- [Museen und vergleichbare Kultur- und Bildungseinrichtungen](museum/index.md)
- [Parks, Zoos, Tiergärten, Freizeitparks](park/index.md)
- [Theater, Opernhäuser, Konzerthäuser](theater/index.md)
- [Temporäre Großveranstaltungen wie Gartenschauen, Volksfeste, Festivals](event/index.md)
